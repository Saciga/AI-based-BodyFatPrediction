# Body Fat Prediction App - Frontend

This project is a React.js application that serves as the frontend for the Body Fat Prediction App. It allows users to input their body measurements and receive predictions for their body fat percentage, along with health recommendations.

## Features

- User-friendly form for inputting body measurements.
- Displays predicted body fat percentage.
- Provides health category, risk level, and personalized recommendations based on the prediction.
- Responsive design for better user experience.

## Getting Started

To run the frontend application, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd bodyfat-prediction-app/frontend
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Start the development server:**
   ```
   npm start
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000` to view the application.

## Folder Structure

- **public/**: Contains static files like `index.html` and `manifest.json`.
- **src/**: Contains the React components and styles.
  - **components/**: Contains reusable components like `Form.js` and `Result.js`.
  - **styles/**: Contains CSS files for styling the application.

## Dependencies

This project uses the following libraries:

- React
- ReactDOM
- Axios (for making HTTP requests to the backend)
